Getting Started

Download Client

Once the download has completed, run overtrack.exe

Depending on your system security you may be warned by Windows SmartScreen, or other such similar security system, that overtrack.exe has been prevented from running - in the case of Windows SmartScreen, click on More info, then, Run anyway to proceed. This only needs to be done once.

OverTrack runs directly from this launcher, so make sure the executable is stored somewhere where it won't be deleted (e.g. on your desktop or in your documents folder).
By default OverTrack will not launch on startup - if you would like OverTrack to start automatically, right click on the icon in the notifications tray and check "Start OverTrack when Windows starts"

Upon first launch of the client you will be asked to authenticate/pair the client to your account using the OverTrack site. This should automatically open a page in your browser where you can click the Authenticate Client button:

If you are not logged in you will instead be presented with the welcome page - simply use the sign-in button and you will be redirected to the authenticate page.

Using OverTrack

OverTrack has a taskbar icon showing the current state and will present toasts (only visible if Overwatch is not fullscreen):



To ensure SR is tracked correctly you must visit the Play Menu (where the Quick Play, Arcade, Competitive Play, and, Play vs. AI buttons are seen) BEFORE and AFTER each game, as prompted by the toasts.

For best results for tracking scores, let the tracker see all score screens - this means not tabbing out or leaving the game at the round/game end score screens.

To close the client, right click the tray icon and select Quit

Is This Allowed?

Blizzard has confirmed that they don't consider OverTrack a cheat and are fine with it.
OverTrack behaves just like any other screen recording software e.g. OBS, Shadowplay. It does not interact with Overwatch and does not inject, modify or read Overwatch's memory, code or data files.

Troubleshooting

If you seen an error "Failed to execute script wrapper" when launching you may need to download and install vc_redist.x64.exe from Microsoft

Notes:

Fullscreen tracking is not available in Windows 7.

In order to help with bugs and development the client will upload logs and occasionally full screenshots of Overwatch. For normal operation it will only upload small sections of the screen, but, when the client encounters oddities, these testing builds can upload more. This means that, for now, images of your whispers may be uploaded and the uploaded logs may contain your username.
