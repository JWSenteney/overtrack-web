FAQ / Troubleshooting

For general troubleshooting or if your question is not answered here please join the OverTrack Discord.

Is OverTrack a cheat?

Blizzard's current stance on third party programs is unclear. Some third party programs with similar functionality to OverTrack have been forbidden.
You may wish to discontinue use of OverTrack until its legality is resolved. Get more details and keep up to date on this issue in the OverTrack Discord

OverTrack behaves just like any other screen recording software e.g. OBS, Shadowplay. It does not interact with Overwatch and does not inject, modify or read Overwatch's memory, code or data files.
Data is only uploaded after a game has completed, so OverTrack has no way of providing an advantage while in game.


How do I solve 'Failed to execute script wrapper' when launching OverTrack?

Download and install vc_redist.x64.exe from Microsoft.
This issue can also be caused if you have non-english characters in your username.
If the problem persists join the OverTrack Discord and ask for help there.


Where does OverTrack install to?

OverTrack runs directly from the .exe file you downloaded. To launch manually simply run the .exe or the desktop shortcut OverTrack creates when it first runs.
OverTrack also creates a config.json file in "%appdata%\overtrack", which can be deleted to unlink your account and places a copy of the executable here.
To uninstall OverTrack just uncheck 'Start OverTrack when Windows starts' from OverTracks menu in the taskbar (and optionally delete the executable).


OverTrack is not picking up Overwatch

OverTrack can have difficulty picking up Overwatch for the following reasons:

  • Overwatch is being run as admin, but OverTrack is not.
  • Overwatch is being run in fullscreen mode on Windows 7. Windows 7 fullscreen capture requires using OBS Capture.
  • You are using a laptop with switchable graphics and running Overwatch on the dedicated GPU. Laptop fullscreen capture requires using OBS Capture.
  • Overwatch is being run in 21:9 aspect ratio (ingame setting).
  • You have an in-game overlay (e.g. ShadowPlay overlay, Discord Overlay, Windows registration text). Either disable this overlay or use OBS Capture.
  • Overwatch is not set to English.
  • The killfeed is disabled.
  • You are using an unusual brightness/contrast or colorblind mode.

How does OverTrack work?

OverTrack applies Computer Vision and Machine Learning techniques to analyse and extract information from what is displayed onscreen. This allows OverTrack to track information such as match SR, map, killfeed and tab-screen statistics not directly provided by Blizzard.


OBS Integration

OverTrack can make use of OBS's capture, provided you have a Game Capture for Overwatch active in the current scene.
This option offers the minimum performance impact of all capture methods and has the least chance of causing stutter.
Additionally OBS capture allows recording on Windows 7, Laptops with switchable graphics and from Overwatch when alt-tabbed (other fullscreen capture methods only work when Overwatch is in the foreground).

To use this feature, make sure you are running OBS (64 bit) or OBS Streamlabs and capturing from Overwatch using a Game Capture source and enable "Capture from OBS".

Note that:

  • If OBS is being run as admin, OverTrack must be as well.
  • The capture source must be enabled and in the current scene. The scene preview can still be disabled.
  • Because the capture must be in the current scene, if you are streaming and switching scenes you may need to add the Game Capture source to the other scenes (you can drag it outside of the scene canvas or make it small and hide it behind another source).
  • If you are not streaming and want the minimum possible performance impact, set OBS's FPS to 4, limit the game capture FPS and disable the preview.
  • If you are using a laptop with switchable graphics, both OBS and OverTrack must also be run on the GPU (either through right click > Run with Graphics Processor or with the Nvidia control panel) or OBS must be using the Game Capture setting "SLI/Crossfire Support".

Battletag Reading

OverTrack can read your Battletag from the Blizzard App to improve name recognition.
This feature must be enabled manually, but is recommended provided you are prepared for OverTrack to read from the Blizzard App's memory (OverTrack still does not and will never read from Overwatch's memory as this would classify it as a cheat).
Additionally you should only enable this option if you switch accounts using a single Blizzard App. If you run multiple apps, or switch accounts in-game this method will cause incorrect results.

OverTrack is causing stutter/frame drops

Stutter/frame drops can be caused by OverTrack's fullsceen capture method - specifically if you are using multiple monitors and playing media on a secondary monitor while playing Overwatch, although it can also occur without this.
Using OBS Capture will mitigate this issue.


OverTrack is missing my SR for games

In order to track SR, make sure you are visiting the play menu before and after each game so OverTrack can capture the SR. If OverTrack missed the SR for a game you can manually provide it using the 'Edit' button on games.


I have a game showing up under the wrong account/in another tab in the games list

Sometimes OverTrack can have trouble working out the name of a player that a game belongs to. This can happen if you were alt-tabbed out when the game started. To fix games with the wrong player name you can use the 'Edit' button to change the account name a game is associated with.


My games/some of my games show up under an account beginning with @

OverTrack is unable to read special characters in player names - if your account has special characters your games will show up under '@battletag'.
You can also let OverTrack use the battletag from your currently logged in Battle.net account by enabling "Read Battletag From Blizzard App"
. If your account does not have special characters this could be caused by OverTrack having difficulty detecting the correct player name. Use the 'Edit' button to correct these games, and report the issue through discord if this happens frequently.


Why are some players @UNKNONWN_BLUE#0 or @UNKNONWN_RED#1?

@UNKNOWN names appear for players who use special characters as part of their battletags.
