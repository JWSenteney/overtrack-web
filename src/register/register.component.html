Public beta

OverTrack is beta software. This means there will be bugs.
If you encounter any oddities, bugs or have queries/suggestions please join the OverTrack Discord and report them in #testing-and-feedback.

Caveats

To track games OverTrack records your screen and uses computer vision to detect kills, scores, SRs and mode.
This means OverTrack uses a small amount of CPU usage in the background - for mid-high end PCs this is not an issue but for low end PCs OverTrack can significantly decrease FPS or even fail to record entirely.
Generally, the performance impact of OverTrack is less than that of OBS, but this may vary.

Note that on Windows 7 OverTrack is unable to track Overwatch when it is running in fullscreen mode. Overwatch must also be run in English with the killfeed enabled.

Using OverTrack

In order to correctly track everything correctly, OverTrack needs to see specific screens in Overwatch.
To track games, try to ensure that OverTrack sees the teams list at the start of each competitive game. If Overwatch is minimized while you are queued then OverTrack may be unable to see the player list - prefer using Alt-Enter to toggle fullscreen mode between games than Alt-Tabbing away from the game when in fullscreen mode. To track SR you need to visit the 'Play Menu' (where the SR is visible below 'Competitive') between each game, and allow OverTrack a second to record the SR.

To check OverTrack is picking up Overwatch hover over the taskbar icon to see OverTrack's status.

There is no need to sign in with multiple accounts, once you have authenticated different accounts will show under separate tabs.
